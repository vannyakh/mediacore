# Roadmap

Canonical roadmap with status: [`docs/getting-started/roadmap.md`](docs/getting-started/roadmap.md).

## Summary

### v0.1 — Foundation

- Core runtime, plugin loader, CLI, REST API, SDK surface

### v0.2 — Platform surfaces

- Dashboard, event system, pipeline hardening, workers, storage plugins

### v0.3 — Desktop, Studio & quality

- Tauri desktop, visual Studio workflows, benchmarks, experimental marketplace

### v1.0 — Stable ecosystem

- Stable API/SDK, production docs, long-term support

Guiding principles: core first, plugins for everything else, SDK consistency, compliance (official/permitted access only).
